# First Preferences 2025 Australian Election Model

## How to run
1. Clone the repo
2. Ensure required CSV files are in `data/raw/`
3. Run:
   python Simulate_Model_from_Fundamentals_Polling.py

## Notes
This repository contains research-grade code intended to be run end-to-end.
Intermediate files are written to `data/generated/` and ignored by git.
